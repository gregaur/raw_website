Beauty Review
Branding, UX/UI, Front-End Development

Beauty Review were looking to add some new functionality and tidy up an existing site that was in need of help.

The existing site had been in operation for a number of years and was looking very tired after a number of hasty additions.

I created numerous concept designs for the client using their existing brand colours as they weren't exactly sure what they were looking for. Eventually, after much back and forth, we found what the client was looking for!

We also built some new photowall functionality for the site which enables users to upload their own photos. My role on this project included UX/UI as well as front-end development.

User Experience issue

Users were arriving at the website uncertain what the point of the site was or what they should do next. After discussion and feedback from users and client, we came up with a more expansive menu system which provides more detail and direction, giving users a purpose upon arriving at the site.

Freshening up the brand.

A large part of this project was freshening up the branding. The old site had an overload of pink and quite a few low-res elements which made the site rather low budget. I revisited the online colours and added a slight purple tinge to the blacks so they would complement the pinks.
I also took out some of the texture a previous designer had implemented. This helped the imagery stand out against the flat colour.
