Bachcare Mobile

Role: UI Design
  • Project lead: Boris Hasler
  • UX: PurpleShirt
  • Back-end development: Sam Walsh

This project was more about tweaking and refining an existing user interface that wasn't quite working for the client. I worked closely with Blake Lough from Purpleshirt to refine a number of elements including bringing a slight gradient to the header and footer to give it some depth and a number of tweaks to typography spacing as well as adjusting gradients and buttons. The client was happy with the result and I felt we managed to achieve a design that enhances the Bachcare brand
