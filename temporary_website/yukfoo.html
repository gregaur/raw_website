Yukfoo

Role: UX/UI Design / Front-end Development
  • Project lead: Boris Hasler
  • Design: Stacey Eyles | Greg Rawlinson
  • Back-end development: Sharne Allen

Yukfoo are a specialist animation company who wanted a fully responsive custom website to showcase their videos and still imagery. Initially they had a couple of pages designed by their graphic designer (Stacey), which set the tone for much of the design work. They were after a layout which would give them the flexibility to add imagery and videos to create a montage type effect. I came up with the solution of enabling the admin user to create a 'split tile' in the cms which would split the tile they're uploading into two parts, for two separate images (see the split tile grid below). This broke up the standard grid pattern nicely on parts of the site and gave the client further creative input into their website. My responsibilities for this project spanned from UX/UI Design, interaction animations to creating the full front-end functionality of the website.
